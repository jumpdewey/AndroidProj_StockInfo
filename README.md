# AndroiProj_StockInfo
　　　1Using Background Thread to do network access in order to let UI keep motivated:) I have tried Runnables as well as AsyncTask to handle the background thread. 
　　　2Retrieving information from Internet by using URL class and HttpURLConnection client.
　　　3Decoding infomation retrieved from the Internet. Generally, there are 3 common formats: XML, JSON, SOAP. In this case, I choose JSON format information to access. A JSON Object is a modifiable set of name/value mappings. I used built-in JSONObject to parse json data.
